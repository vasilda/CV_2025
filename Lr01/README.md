## <b> 1. Теоретическая часть </b>
<b> Фильтр Собеля </b> - это оператор для выделения границ на изображениях, который вычисляет приближенное значение градиента яркости. Он основан на свертке изображения с двумя отдельными ядрами 3×3 - для горизонтальных и вертикальных производных.

<i> Горизонтальное ядро (Gx): </i>

|     -1      |     0      |     1      |
|-------------|------------|------------|
| <b> -2 </b> | <b> 0 </b> | <b> 2 </b> |
| <b> -1 </b> | <b> 0 </b> | <b> 1 </b> |

<i> Вертикальное ядро (Gy): </i>

|      1      |      2      |      1      |
|-------------|-------------|-------------|
|  <b> 0 </b> |  <b> 0 </b> |  <b> 0 </b> |
| <b> -1 </b> | <b> -2 </b> | <b> -1 </b> |

<i> Градиент в каждой точке: </i> 

`G = √(Gx² + Gy²)`

 
Результатом применения оператора Собеля является двумерная карта градиента для каждой точки. Её можно обработать и показать как картинку, на которой участки с большой величиной градиента (в основном, грани) будут видны как белые линии. 

## 2. Описание разработанной системы
<b> Реализации: </b>
1. Библиотечная реализация (Scikit-Image): filters.sobel (поиск всех границ), filters.sobel_v (поиск вертикальных границ), filters.sobel_h (поиск горизональных границ);
2. Нативная реализация (Python + NumPy).

<b> Принципы работы (для нативной реализации): </b>
1. Загрузка изображения и преобразование в grayscale;
2. Паддинг для изображения;
3. Применение ядер свертки для вычисления производных;
4. Вычисление магнитуды градиента;
5. Сравнение производительности различных методов.
